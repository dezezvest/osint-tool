🔍 Инструмент для поиска по множеству баз данных (Telegram, банки, утечки)

Возможности

- Поиск по номерам телефонов
- Поиск по именам
- Поиск по email
- Поиск по username
- Поиск по любому слову
- Поддержка CSV, XLSX, TXT
- Показывает прогресс поиска

Установка

```bash
git clone https://github.com/dezezvest/osint-tool.git
cd osint-tool
pip install -r requirements.txt


Установка баз данных

Скачай archives.zip с релиза
wget https://github.com/dezezvest/osint-tool/releases/download/v1.0.0/databases.zip
Или скачай вручную по ссылке: https://github.com/dezezvest/osint-tool/releases
Распакуй архив в папку data
unzip databases.zip -d data/
